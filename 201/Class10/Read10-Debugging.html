What Went Wrong? Troubleshooting JavaScript

1.) Name some key differences between a Syntax Error and a Logic Error.
Syntax errors are caused by mistakes in the syntax of the code like a missing bracket or semicolon preventing the code from running at all, Logic errors are mistakes in the actual logic or flow of the code causing unexpected or incorrect results but still allowing the code to run

2.) List a few types of errors that you have encountered in past lab assignments and explain how you were able to correct them.
One type of error I've encountered is a TypeError which happens when trying to perform an operation on a variable of the wrong type. To correct it I had to check the type of the variable and make sure it matched the operation I was trying to perform

3.) How will this topic continue to influence your long term goals?
it'll save me time and frustration when I get errors, and improve my problem-solving and critical thinking abilities

The JavaScript Debugger

1.) How would you describe the JavaScript Debugger tool and how it works to someone just starting out in software development?
It's a tool that allows developers to step through their code one line at a time, set breakpoints, and monitor variables/values as the code executes

2.) Define what a breakpoint is.
A point in the code where the execution pauses letting the developer inspect the state of the program and variables at that point

3.) What is the call stack?
List of function calls that have been made in the current program
