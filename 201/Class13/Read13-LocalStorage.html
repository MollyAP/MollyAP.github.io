Local Storage

1.) Why would a developer use local storage for a web application?
It's useful for applications that need to retain data between sessions or across different pages within itself.

2.) What information should not be stored in local storage?
Passwords, tokens, personal info etc

3.) Local storage can store what type of data? How would you convert it to that type before storing? string (numbers, objects, or arrays with conversions). you can convert to a string by using the "JSON.stringify()" method before storing them in local storage.
