Images in HTML

In order to put an image on a web page, we use the < img> element, it can't have any child content or an end tag and needs two attributes to be useful, src and alt

src contains a URL pointing to the image you want to embed, like href for < a >, src can be a relative or absolute, Without an src attribute, an img element has no image to load.

So for example, if your image is called test.png, and it sits in the same directory as your HTML page, you could embed the image like so: <img src="test.png" alt="Test img" />
If the image was in an images subdirectory, which was inside the same directory as the HTML page, then you'd embed it like this: <img src="images/test.png" alt="Test img" />
Though not reccomended you can even embed the image using its absolute URL like this: <img src="https://www.example.com/images/test.png" alt="Test img" /> Hosting your own images by keeping them on the same server as your HTML is preffered but In more advanced setups, you might use a CDN (Content Delivery Network) to deliver your images.

Colors in CSS

color: defines the foreground color of an HTML element's content

background-color: defines the element's background color

Text

Whenever an element is rendered, these properties are used to determine the color, background,` and any decorations of the text

color: The color to use for text and text decorations (under or overlines, strike-through lines, etc)

background-color: The text's background color

text-shadow: Puts a shadow effect on text, you can also change the shadow's base color, which is then blurred and blended with the background based on the other parameters

text-decoration-color: text decorations use the color property as their colors by default but you can override that and use a different color for them with the text-decoration-color property

text-emphasis-color: color used when drawing emphasis symbols adjacent to each character in the text, used primarily with text for East Asian languages.

caret-color: color used for a caret (the text input cursor) within the element. This is only useful in elements that are editable, such as < input > and < textarea > or elements whose HTML contenteditable attribute is set.

boxes: every element is a box with content, and has a background and a border in addition to whatever contents the box may have.

borders: allows you to edit the property of the box's outline

background-color: color to use in areas of the element that have no foreground content

column-rule-color: color used when drawing the line separating columns of text

outline-color: color used when drawing an outline around the outside of the element, outlines are different from borders since they dont get space set aside for them in the document meaning overlapping other content is possible, it's generally used as a focus indicator, to show which element will receive input events.
