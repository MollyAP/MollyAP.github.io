CSS Transitions & Animations

1.) What does a CSS transition allow the developer to do to an element?
Apply a smooth transition effect to it when its state changes like when someone hovers over it or clicks on it.

2.) How does a CSS animation differ from a CSS transition?
Animations can have multiple stages and can be looped continuously or paused at certain points.
