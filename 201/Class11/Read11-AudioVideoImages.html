Video and Audio Content

1.) Explain how the ability to use video and audio on the web has evolved since the early 2000s.
The ability to use video and audio on the web has evolved significantly since the early 2000s. In the past, plugins like Flash were used to display video and audio content, but now browsers support HTML5 video and audio tags, making it easier to embed media directly on web pages. Additionally, improvements in internet speeds have made it possible to stream high-quality video and audio content without buffering.

2.) Describe the use of the src and controls attributes in the < video> element.
The src attribute is used to specify the URL of the video file, while the controls attribute enables standard video playback controls, such as play, pause, and volume.

3.) Why is it important to have fallback content inside the < video> element?
Because not all browsers support the < video> tag or certain video codecsand by providing fallback content users with unsupported browsers or codecs can still access the content.

A Complete Guide To Grid

1.) How does Grid layout differ from Flex?
Flexbox is designed for one-dimensional layouts like rows or columns and Grid is designed for two-dimensional layouts with rows and columns.

2.) Grid container, grid item, and grid line are a few important terms to understand when using Grid. Please describe these terms in a few sentences.
The grid container is the element that holds all the grid items which are the individual elements inside the container that are laid out using the grid system and grid lines are the horizontal and vertical lines that define the rows and columns of the grid layout.

Responsive Images

1.) Besides making a site visually appealing across different screen sizes, why should developers make images responsive?
To improve site performance and user experience as Large high-resolution images can slow down page load times.

2.) Define the following < img> attributes srcset and sizes. Write an example of how they are used.
The srcset attribute is used to specify multiple versions of an image at different resolutions, while the sizes attribute specifies the width of the image container. For example, specifies two versions of the image with widths of 480px and 800px, and tells the browser to use the smaller version on screens up to 600px wide.

3.) How is srcset more helpful for responsive images than CSS or JavaScript?
Because it allows the browser to select the appropriate image based on the device's screen size and resolution.
