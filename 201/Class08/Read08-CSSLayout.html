Class Notes

Learn CSS - Flexbox

1.) Flexbox is designed for one-dimensional content. Explain what this means.
It works along a single axis - either horizontally or vertically

2.) Explain the difference between the main axis and cross axis.
The main axis is the primary axis that items are laid out along, the cross axis is the axis that runs perpendicular to the direction of the main axis

3.) How can using certain properties of flexbox negatively impact accessibility?
Some things can cause confusion for screen reader users, while others can cause issues for users who rely on keyboard navigation

CSS Layout - Flexbox

1.) What are some advantages of using flexbox over float?
Easier and more intuitive layout, Responsive design, Efficient use of space, Simplifies code, Better browser support
