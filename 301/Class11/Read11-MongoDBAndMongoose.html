sql vs nosql

  1. What does SQL stand for?
    2. Structured Query Language

  2. What is a relational database?
    3. A database that stores and organizes data in a series of tables with rows and columns

  3. What type of structure does a relational database work with?
    4. A structured data model in which data is organized into one or more tables

  4. What is a ‘schema’?
    5. The structure that defines the objects in the database and the relationships between them

  5. What is a NoSQL database?
    6. A database that is designed to handle and store large volumes of unstructured and semi-structured data

  6. How does it work?
    7. By using various data models to store and organize data

  7. What is inside of a MongoDB database?
    8. Collections that contain documents

  8. Which is more flexible - SQL or MongoDB? and why.
    9. MongoDB. It stores data in flexible, semi-structured documents instead of using fixed schemas and tables

  9. What is the disadvantage of a NoSQL database?
    10. Fewer features than SQL databases
