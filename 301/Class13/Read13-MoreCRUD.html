  2. If you had to describe the process of creating a RESTful API in 5 steps, what would they be?
    1. Define your API's purpose and identify the resources you want to expose.
    2. Design the structure of your API endpoints, specifying the URLs and actions for each resource.
    3. Implement the API by writing the code and logic to handle requests and process data.
    4. Thoroughly test and debug your API to ensure it functions correctly and reliably.
    5. Publish and share your API with others, providing clear documentation and instructions for its usage.
