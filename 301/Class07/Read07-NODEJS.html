An Introduction to Node.js on sitepoint.com

  1. What is node.js?
    2. Node.js is an event-based, non-blocking, asynchronous I/O runtime that uses Google’s V8 JavaScript engine and libuv library

  2. In your own words, what is Chrome’s V8 JavaScript Engine?
    3. The V8 engine is a JavaScript engine that runs on Chrome designed with performance in mind, compiling JavaScript directly to native machine code that your computer can execute

  3. What does it mean that node is a JavaScript runtime?
    4. it's a program used to execute JavaScript on computers

  4. What is npm?
