    2. A function invocation

  2. How many ‘calls’ can happen at once?

  3. What does LIFO mean?

  5. What causes a Stack Overflow?

JavaScript error messages

  1. What is a ‘reference error’?
    2. when you try to use a variable that isn't declared yet

  2. What is a ‘syntax error’?
    3. when you have something that's syntax can't be parsed

  3. What is a ‘range error’?
    4. when you give an invalid length to an object with some kind of length you're trying to manipulate.

  4. What is a ‘type error’?
    5. when the types you are trying to use or access are incompatible

  5. What is a breakpoint?
    6. the point in which your code breaks

  6. What does the word ‘debugger’ do in your code?
    7. pauses the execution of the code at a specific point so you can inspect it in the debugger.
