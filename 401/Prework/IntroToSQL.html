Class Notes
Relational databases provide a structured way to organize and store data using tables with rows and columns. Through these many many exercises, I learned how tables can be related to each other through common columns, establishing connections between data entities. This enables efficient data retrieval, manipulation, and querying.
SQL, or Structured Query Language, serves as the language for interacting with relational databases. I now understand how to use SELECT statements to retrieve data from tables based on specified conditions, apply filtering using the WHERE clause, and sort results using the ORDER BY clause. I also learned how to work with relational databases and manipulate data effectively using SQL commands.
