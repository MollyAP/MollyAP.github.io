Trees - Vocabulary/Definition

  • Node: A component of a tree that may contain its own values and references to other nodes.
  • Root: The initial node of the tree.
  • K: A number specifying the maximum number of children a node can have in a k-ary tree.
  • Left: Reference to the left child node in a binary tree.
  • Right: Reference to the right child node in a binary tree.
  • Edge: Link between a parent and child node.
  • Leaf: A node with no children.
  • Height: Number of edges from the root to the furthest leaf.
